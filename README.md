# Floresta Protegida Portal

## Usage 

```bash
npm install
npm run dev
# or
yarn
yarn dev
```

## Deploy

### Dokku (preferred)

`git remote add dokku dokku@encenar.tk:florestaprotegida
`git push dokku master`


### Now

Deploy it to the cloud with [now](https://zeit.co/now) ([download](https://zeit.co/download)):

```bash
now
```
